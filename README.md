# 🌳 TradeTree

منصة تحليل الأسهم الأمريكية بالذكاء الاصطناعي

## الإعداد

1. افتح الموقع
2. اضغط ⚙️ الإعدادات
3. أضف **Twelve Data API Key** (مجاني من [twelvedata.com](https://twelvedata.com))
4. أضف **Claude API Key** اختياري (من [console.anthropic.com](https://console.anthropic.com))

## المميزات

- 📊 تحليل الأسهم الأمريكية
- 🧠 تحليل ذكي بـ Claude AI
- 📈 رسوم بيانية TradingView
- ⭐ قائمة المتابعة
- 💼 تتبع المحفظة
- 📅 التقويم الاقتصادي

## تنبيه

المعلومات لأغراض تعليمية فقط. لا تشكل توصية استثمارية.
